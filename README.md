# 🚀 Ultra Neon Dark Dashboard

A modern **Ultra Neon Dark Dashboard** built using **HTML5** and **CSS3**. This project features glowing neon borders, animated effects, responsive grid layouts, and a clean futuristic UI without using JavaScript.

---

## ✨ Features

- 🌙 Modern Dark Theme
- 💙 Cyan & Pink Neon Glow Effects
- ✨ Animated Rotating Borders
- 📱 Fully Responsive Grid Layout
- 🎨 Glassmorphism Style Cards
- ⚡ Smooth Hover Animations
- 🔥 Pure HTML & CSS
- 🎯 Font Awesome Icons

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
- CSS Variables
- CSS Animations
- Font Awesome

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/TineshChasiya/Ultra-Neon-Dark-Dashboard.git
```

2. Open the project folder.

3. Open `index.html` in your browser or run it using **Live Server**.
